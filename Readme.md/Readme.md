# 🍔 Tasty-Tasty Website

A beginner-friendly multi-page food website created using **HTML** and developed using **Visual Studio Code**.

This project is made for learning basic web development and understanding how to create and link multiple web pages.

---

## 📖 Project Overview

**Tasty-Tasty** is a simple static website for a restaurant/food service.  
It includes multiple pages like Home, Menu, and About, connected using navigation links.

This project helps beginners understand:
- Website structure  
- Page navigation using links  
- Folder organization  

---

## 🛠️ Technologies Used

- HTML 
- Visual Studio Code  
- Git & GitHub
